La librairie javascript Acte.js permet de saisir et manipuler des dates grégoriennes, républicaines et juliennes.

Comment utiliser la librairie JavaScript ?

Avec Node.js

$ npm install acte --save

Dans une page html

<script type="text/javascript" src="acte.min.js"></script>

Convertir une date

La zone de saisie accepte les abbréviations courantes (an V, Xbre, Jet, etc.).
