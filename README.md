## Description
Librairie Javascript pour manipuler des données généalogiques

## Installation

### Pour Node.js :
```
$ npm install acte --save
```

### Pour le navigateur :
Téléchargez et décompressez la dernière archive zip ([acte-0.0.9-3-dist.zip](https://github.com/gtoubiana/acte/blob/master/dist/acte-0.0.9-3-dist.zip?raw=true)), puis appelez le script `acte.min.js` au début ou à la fin de votre page html.
```html
<script type="text/javascript" src="acte.min.js"></script>
```
